# UbiDemo
LIVE - https://evening-lake-70202.herokuapp.com/

## Download and Run
---

git clone https://github.com/joaoA/UbiDemo.git

cd UbiDemo/UbiFrontEnd

npm install

ng build --env=prod

cd ../UbiwhereDemo

mvn clean install

mvn spring-boot:run




## Project Info
---

UbiFrontEnd  => Angular 4 + bootstrap

UbiwhereDemo => Spring boot + JPA + postgres
